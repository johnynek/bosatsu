# Bosatsu

Bosatsu language support for Visual Studio Code.

## Features

- Syntax highlighting for `.bosatsu` files
- Keywords, functions, structs, enums, and comments
- Numeric literals and character literals (`."x"` / `.'x'`)
- String interpolation highlighting for `${...}` and `$.{...}`

## Usage

Install the extension, then open any `.bosatsu` file.

## Extension Settings

This extension does not add any settings.

## Known Issues

- Highlighting of types/constructors is capitalization-based and may over-highlight in uncommon cases.
- This extension provides syntax highlighting only (no semantic tokens yet).
