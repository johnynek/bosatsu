## Plan Template

```markdown
# Feature Name

**Status:** PLANNING | IN_PROGRESS | COMPLETE | ABANDONED
**Branch:** feature-branch-name
**Issue/PR:** #123

## Overview
Brief description of what this accomplishes.

## Design Decisions
Key choices and tradeoffs.

## Implementation Checklist
- [ ] Task 1
- [ ] Task 2

## Open Questions
Things still to be decided.
```
